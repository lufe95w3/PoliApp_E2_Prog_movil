PoliApp

PoliApp es una aplicación móvil desarrollada con Ionic Framework y Angular. Su objetivo es ofrecer información sobre las carreras y programas académicos de la universidad, facilitando la consulta para estudiantes y aspirantes.

Características
Menú principal con acceso a secciones clave.
Información detallada sobre las carreras y programas académicos.
Fechas importantes de inscripciones.
Sección de contacto para resolver dudas.
Diseño responsive y adaptado para dispositivos móviles.
Tecnologías utilizadas
Ionic Framework (para el desarrollo de aplicaciones híbridas).
Angular (para el manejo de componentes y lógica).
TypeScript (lenguaje principal).
CSS/SCSS (para el estilo).
Node.js y npm (para la gestión de dependencias).
