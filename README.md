# **PoliApp** 📚🎓

**PoliApp** es una aplicación móvil desarrollada con **Ionic Framework** y **Angular**, diseñada para facilitar el acceso a información sobre carreras y programas académicos de la universidad. Está dirigida tanto a estudiantes como a aspirantes que buscan información clara y rápida desde sus dispositivos móviles.

---

## **Características** 🚀

- 🏠 **Menú principal**: Acceso rápido a las secciones clave de la aplicación.
- 📄 **Información académica**: Detalles completos sobre las carreras y programas ofrecidos por la universidad.
- 📅 **Fechas importantes**: Información actualizada sobre inscripciones y eventos relevantes.
- 📞 **Sección de contacto**: Resuelve dudas y recibe soporte a través de un formulario de contacto.
- 📱 **Diseño responsive**: Optimizado para dispositivos móviles, garantizando una experiencia de usuario fluida.

---

## **Tecnologías utilizadas** 🛠️

- **Ionic Framework**: Para el desarrollo de aplicaciones híbridas multiplataforma.
- **Angular**: Para la gestión de componentes y lógica de la aplicación.
- **TypeScript**: Lenguaje principal para un código robusto y tipado.
- **CSS/SCSS**: Diseño visual y estilos personalizados.
- **Node.js y npm**: Gestión de dependencias y herramientas de desarrollo.
